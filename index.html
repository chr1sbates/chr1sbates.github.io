The Fat Neek URL Shortener
2022

During a late night crusade, I got very bored and teamed up with a friend to create a URL shortener. The Fat Neek URL Shortener is a very basic shortener which has a modern look and feel. The shortener gives the user control of both the site and the link generated allowing you to choose a dark or light mode and also allowing the user to give a name to the shortened link so the ID isn't randomly generated
